# 📄 ATS Resume Analyzer

An ATS (Applicant Tracking System) Resume Analyzer built using **Node.js**, **Express.js**, and **JavaScript**. The application extracts text from PDF resumes, performs keyword-based skill matching, and helps users identify missing skills to improve their resumes for job applications.

---

# 🚀 Project Overview

ATS Resume Analyzer is a web application that simulates the basic functionality of an Applicant Tracking System (ATS). Users can upload a PDF resume, and the application extracts its content, compares it against predefined technical skills, and provides instant feedback on matched and missing keywords.

This project demonstrates backend development, PDF processing, file handling, and REST API implementation using Node.js and Express.js.

---

# ✨ Features

- 📄 Upload resumes in PDF format
- 🔍 Extract text from PDF files
- 📊 Keyword-based ATS skill analysis
- ✅ Display matched skills
- ❌ Highlight missing skills
- ⚡ Fast and lightweight backend
- 🎨 Responsive user interface

# ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/ashuya0304/Resume-analyzer.git
```

### Navigate to the project

```bash
cd Resume-analyzer
```

### Install dependencies

```bash
npm install
```

### Start the server

```bash
node server.js
```

### Open your browser

```
http://localhost:3000
```

# 📖 Usage
